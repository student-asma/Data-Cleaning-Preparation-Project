# Data Cleaning & Preparation using Microsoft Excel

## Project Overview

This project was completed as part of my Data Analytics Internship at DecodeLabs.

The objective was to clean and prepare a raw dataset by identifying missing values, checking duplicate records, validating data formats, and ensuring overall data consistency.

## Tools Used

* Microsoft Excel

## Tasks Performed

* Identified missing values
* Replaced missing CouponCode values with "No Coupon"
* Checked duplicate Order IDs
* Verified date formats
* Validated Quantity, UnitPrice, and TotalPrice
* Ensured consistent values in Product, PaymentMethod, OrderStatus, and ReferralSource columns

## Outcome

The dataset was successfully cleaned and prepared for further data analysis.
